# 种子数据使用指南

## 快速开始

1. **确保数据库连接**
   确保您的 `.env` 文件中有正确的 `DATABASE_URL` 配置。

2. **运行种子脚本**
   ```bash
   npm run seed
   ```
   或
   ```bash
   pnpm seed
   ```

3. **验证数据**
   种子脚本运行成功后，您将看到详细的创建日志。

## 登录测试账户

运行种子脚本后，您可以使用以下账户登录系统：

- **管理员**: `admin@example.com` / `123456`
- **组织者**: `organizer@example.com` / `123456`

## 注意事项

- 种子脚本会**清空现有数据**，请在开发环境中使用
- 如果遇到错误，请检查数据库连接和Prisma配置
- 所有用户密码都是 `123456`

## 生成的数据内容

种子脚本会创建：
- 5个用户（不同角色）
- 5个参与者
- 1个活跃比赛
- 4个评分标准
- 5个节目（不同状态）
- 36个评分记录
- 3个排名记录
- 3个审计日志
- 3个文件记录

详细信息请查看 `prisma/seed-data-description.md`。
