## 🎯 核心功能特性

### 多选手节目支持
- **分隔符识别**: 支持逗号（,）、中文逗号（，）、顿号（、）
- **自动创建**: 如果选手不存在，会自动创建新选手
- **智能关联**: 自动建立选手与节目的关联关系

### 字段识别算法
- **模糊匹配**: 支持多种列标题命名方式
- **中英文兼容**: 同时支持中文和英文字段名
- **大小写不敏感**: 忽略大小写差异

### 数据验证
- **必填字段检查**: 确保姓名、节目名称等必填字段存在
- **格式验证**: 检查数据格式的正确性
- **重复检测**: 防止重复数据导入

### 用户体验
- **拖拽上传**: 支持文件拖拽上传
- **实时预览**: 上传后立即显示解析结果
- **进度指示**: 显示导入进度和状态
- **错误提示**: 详细的错误信息和解决建议

## 🚀 使用方法

### 导入选手信息
1. 进入"选手管理"页面
2. 点击"批量导入"按钮
3. 下载模板或直接上传Excel文件
4. 预览解析结果
5. 确认导入

### 导入节目信息
1. 进入"节目管理"页面或比赛详情的"节目管理"标签
2. 点击"批量导入"按钮
3. 上传包含节目信息的Excel文件
4. 系统自动处理多选手关联
5. 确认导入

### 多选手节目示例
```csv
节目名称,描述,顺序,选手姓名
《茉莉花》,合唱表演,6,"张三,李四,王五"
《民族风》,团体舞蹈,7,"王五，赵六，钱七"
```
