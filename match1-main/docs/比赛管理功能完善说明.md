# 比赛管理功能完善说明

## 概述

对比赛管理系统进行了全面完善，新增了数据统计分析和数据导出功能，为比赛组织者提供了完整的数据管理和分析工具。

## 新增功能

### 1. 比赛统计分析 (CompetitionStats)

#### 功能特点
- **实时数据统计**: 动态获取比赛进行状态和数据分析
- **多维度统计**: 包含节目、选手、评分、团队等多个维度
- **可视化展示**: 通过图表和进度条直观展示数据
- **分类管理**: 按不同类别组织统计信息

#### 统计内容

##### 概览统计
- 比赛状态和排名模式
- 节目总数和状态分布
- 参与选手数量（团队/个人）
- 评分进度和完成率

##### 节目分析
- 节目状态分布图（等待中/表演中/已完成）
- 完成进度跟踪
- 节目执行统计

##### 评分统计
- 评分概览（最高分、最低分、平均分）
- 评委评分统计
- 评分完成率分析

##### 团队表现
- 团队参与情况
- 团队成员分布
- 团队表现排名

##### 排名榜单
- 实时排名展示
- 奖牌标识（前三名）
- 分数详情

### 2. 数据导出功能 (ExportData)

#### 支持的导出类型

##### 评分数据导出
- **内容**: 所有节目的详细评委打分数据
- **包含字段**:
  - 节目信息（名称、顺序、选手、团队）
  - 评委评分详情（评委ID、分数、评语）
  - 评分标准（名称、权重、最高分）
  - 评分时间记录
- **用途**: 详细的评分分析和审核

##### 排名结果导出
- **内容**: 比赛的最终排名结果
- **包含字段**:
  - 排名顺序
  - 节目和选手信息
  - 总分计算结果
  - 排名更新方式和时间
- **用途**: 成绩公布和存档

##### 选手信息导出
- **内容**: 所有参赛选手的信息
- **包含字段**:
  - 选手基本信息（姓名、团队、联系方式）
  - 参与节目列表
  - 注册时间和个人简介
- **用途**: 选手管理和联系

#### 导出格式

##### CSV 格式
- **特点**: 表格格式，兼容性好
- **用途**: Excel分析、数据处理
- **优势**: 直观、易于编辑和分析

##### JSON 格式
- **特点**: 结构化数据，保持完整性
- **用途**: 程序处理、系统集成
- **优势**: 保留数据结构，支持复杂数据类型

## 技术实现

### API 接口

#### 统计数据接口
```
GET /api/competitions/{id}/stats
```
- **功能**: 获取比赛的完整统计数据
- **返回**: 包含概览、节目、评分、团队等多维度统计
- **特性**: 实时计算、缓存优化

#### 数据导出接口
```
GET /api/competitions/{id}/export?type={type}&format={format}
```
- **参数**:
  - `type`: 导出类型（scores、rankings、participants）
  - `format`: 导出格式（csv、json）
- **功能**: 根据参数生成相应的导出文件
- **特性**: 权限控制、文件命名规范

### 前端组件

#### CompetitionStats 组件
- **位置**: `/src/components/dashboard/competitions/CompetitionStats.tsx`
- **功能**: 统计数据可视化展示
- **特性**: 
  - 响应式设计
  - 分tab组织内容
  - 实时数据更新
  - 错误处理和加载状态

#### ExportData 组件
- **位置**: `/src/components/dashboard/competitions/ExportData.tsx`
- **功能**: 数据导出界面
- **特性**:
  - 多种导出选项
  - 进度状态显示
  - 错误处理和提示
  - 文件自动下载

### 数据处理

#### 统计计算
- **节目状态统计**: 按状态分组计算
- **评分统计**: 加权平均分计算
- **团队统计**: 去重和聚合
- **进度计算**: 百分比和完成度

#### 导出处理
- **CSV生成**: 表格格式化和字符编码
- **JSON生成**: 数据结构保持和序列化
- **文件命名**: 包含比赛名称和日期
- **下载处理**: Blob生成和自动下载

### 数据安全
- **身份验证**: 基于NextAuth的会话验证

- **数据过滤**: 根据用户权限过滤数据

### 文件生成
