# 节目状态控制功能使用说明

## 🎯 功能概述

节目状态控制功能允许管理员和组织者在比赛管理页面的**节目控制台**中实时控制节目状态，方便切换节目后及时更新节目状态，确保大屏幕显示和评分系统的状态同步。

**重要提醒**：选择状态后需要点击"保存节目设置"按钮才能真正保存状态更改，避免误操作。

## 📍 功能位置

**路径**：`比赛管理` → `选择比赛` → `大屏幕管理` → `节目控制台`

具体访问：
1. 登录系统（管理员或组织者账号）
2. 进入 `仪表盘` → `比赛管理`
3. 选择要管理的比赛
4. 点击 `大屏幕管理` 按钮
5. 在页面中找到 `节目控制台` 卡片

## 🎮 功能特性

### 1. 当前节目状态控制

在选择当前显示节目后，可以看到：

- **节目基本信息**：序号、名称、参赛者
- **当前状态标识**：彩色标签显示当前状态
- **状态控制按钮**：
  - ⏰ **等待中** - 节目尚未开始
  - ▶️ **进行中** - 节目正在表演
  - ✅ **已完成** - 节目表演结束

### 2. 智能状态切换

- **智能切换按钮**：一键按顺序切换状态
  - 等待中 → 进行中 → 已完成 → 等待中
- **快速操作**：适合比赛现场快速状态切换

### 3. 节目网格快速控制

- **节目卡片**：每个节目显示为可点击的卡片
- **状态颜色编码**：
  - 🟡 **黄色** - 等待中
  - 🔵 **蓝色** - 进行中  
  - 🟢 **绿色** - 已完成
- **点击选择**：点击节目卡片即可选择该节目

### 4. 批量状态控制

- **全部设为等待中**：重置所有节目状态
- **重置为开始状态**：第一个节目设为进行中，其余设为等待中

## 🔧 使用场景

### 比赛开始前
1. 使用"重置为开始状态"设置初始状态
2. 点击"保存节目设置"按钮确认更改
3. 第一个节目设为"进行中"，其余为"等待中"

### 比赛进行中
1. 当前节目表演结束 → 点击"已完成"
2. 下一个节目开始 → 点击该节目的"进行中"
3. 或使用"智能切换"一键更新
4. **重要**：点击"保存节目设置"按钮确认所有更改

### 状态修正
- 如果误操作，可以点击"清除未保存更改"撤销
- 重新选择正确的状态后保存
- 使用批量控制快速重置状态

## 🎨 视觉反馈

### 状态标识颜色
- **等待中**：黄色背景，时钟图标
- **进行中**：蓝色背景，播放图标  
- **已完成**：绿色背景，对勾图标
- **待保存状态**：标识后显示 `*` 号

### 按钮状态
- **当前状态按钮**：深色高亮显示
- **其他状态按钮**：浅色边框显示
- **禁用状态**：当前状态按钮不可点击

### 保存提示
- **未保存更改**：黄色提示框显示待保存项目数量
- **保存按钮**：显示"保存节目设置 (X 项更改)"
- **成功提示**：绿色成功消息
- **错误提示**：红色错误信息

## 📡 技术实现

### API 端点
- **路径**：`/api/programs/[id]/simple-update`
- **方法**：`PATCH` 或 `PUT`
- **参数**：`{ currentStatus: 'WAITING' | 'PERFORMING' | 'COMPLETED' }`

### 实时更新
- 状态更新后自动刷新节目数据
- 大屏幕显示会同步更新
- 评分系统状态保持一致

### 权限控制
- 仅限**管理员**和**组织者**使用
- 自动验证用户权限
- 未授权访问将被拒绝

## 🔄 状态流转逻辑

```
等待中 (WAITING)
    ↓ 开始表演
进行中 (PERFORMING)  
    ↓ 表演结束
已完成 (COMPLETED)
    ↓ 重新设置
等待中 (WAITING)
```

### 智能切换逻辑
- 从 `等待中` → `进行中`
- 从 `进行中` → `已完成`  
- 从 `已完成` → `等待中`

## ⚡ 快捷操作

### 节目状态操作流程
1. **选择节目**：点击节目卡片或使用上/下一个按钮
2. **选择状态**：点击对应的状态按钮（等待中/进行中/已完成）
3. **确认保存**：点击"保存节目设置"按钮
4. **系统同步**：大屏幕和评分系统自动更新

### 批量操作流程
1. **批量设置**：点击"全部设为等待中"或"重置为开始状态"
2. **检查预览**：查看未保存更改提示
3. **确认保存**：点击"保存节目设置"按钮
4. **完成更新**：所有状态同时生效

### 撤销操作
- **单个撤销**：重新选择正确状态
- **全部撤销**：点击"清除未保存更改"按钮
- **重新开始**：清除后重新设置状态

## 🛠️ 故障排除

### 常见问题

**Q: 选择状态后大屏幕没有变化？**
A: 需要点击"保存节目设置"按钮才能真正保存状态更改

**Q: 如何知道哪些状态还没保存？**
A: 查看黄色的"未保存更改"提示框，显示具体的更改数量

**Q: 误操作了怎么办？**
A: 点击"清除未保存更改"按钮可以撤销所有未保存的状态更改

**Q: 保存按钮显示灰色无法点击？**
A: 确认网络连接正常，刷新页面重试

## 📋 最佳实践

### 比赛现场操作建议
1. **提前测试**：比赛前测试状态控制和保存功能
2. **分步操作**：先选择状态，再确认保存，避免误操作
3. **及时保存**：每次状态变化后及时点击保存按钮
4. **备用方案**：准备手动刷新大屏幕的方案

### 状态管理原则
- **先选择后保存**：避免误操作立即生效
- **批量操作**：多个状态更改可以一次性保存
- **及时确认**：选择状态后尽快点击保存按钮
- **定期检查**：确保未保存更改提示为空
