# 节目管理功能说明

## 概述
新的节目管理页面提供了两种视图模式，让用户可以更灵活地管理比赛节目。

## 功能特性

### 1. 双视图模式
- **按比赛分类视图**（默认）：将节目按比赛分组显示，便于按比赛管理节目
- **列表视图**：传统的卡片网格布局，显示所有节目

### 2. 按比赛分类视图特点
- 可折叠的比赛分组
- 比赛状态显示（进行中、草稿、已结束）
- 实时统计信息：
  - 节目数量
  - 选手总数
  - 各状态节目统计（等待、进行、完成）
- 比赛时间范围显示
- 默认展开第一个有节目的比赛

### 3. 搜索和筛选功能
- 按节目名称或描述搜索
- 按节目状态筛选（等待中、进行中、已完成）
- 在列表视图中可按比赛筛选
- 清除筛选按钮

### 4. 节目操作
- 查看节目详情
- 编辑节目信息
- 状态管理：
  - 等待中 → 开始（进行中）
  - 进行中 → 完成（已完成）
  - 已完成 → 重置（等待中）

### 5. 权限控制
- 管理员可查看所有比赛的节目
- 普通用户只能查看自己组织的比赛或公开比赛的节目
- 节目状态更新需要相应权限

## 技术实现

### 组件结构
```
src/app/dashboard/programs/page.tsx - 主页面
src/components/dashboard/programs/
├── ProgramList.tsx - 传统列表视图
├── ProgramListByCompetition.tsx - 按比赛分类视图
└── 其他相关组件...
```

### API接口
- `GET /api/programs` - 获取节目列表，支持搜索和筛选参数
- `PUT /api/programs/[id]` - 更新节目状态
- `GET /api/competitions` - 获取比赛列表

### 新增UI组件
- `src/components/ui/collapsible.tsx` - 可折叠组件

## 使用说明

1. **切换视图模式**：点击页面上方的"按比赛分类"或"列表视图"按钮

2. **搜索节目**：在搜索框中输入关键词，按回车搜索

3. **筛选节目**：使用状态下拉框筛选特定状态的节目

4. **管理节目状态**：
   - 点击"开始"将等待中的节目设为进行中
   - 点击"完成"将进行中的节目设为已完成
   - 点击"重置"将已完成的节目重置为等待中

5. **查看比赛详情**：在按比赛分类视图中，点击比赛标题可展开/折叠该比赛的节目列表
