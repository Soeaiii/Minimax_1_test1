# 大屏幕节目切换界面优化

## 优化概述

将大屏幕管理页面中的节目切换从卡片网格形式改为列表形式，提供更清晰的界面体验。

## 优化前后对比

### 优化前
- 使用卡片网格布局：`grid grid-cols-3 sm:grid-cols-4 md:grid-cols-5 gap-2`
- 每个节目显示为独立的按钮卡片
- 信息显示较为紧凑，在小屏幕上可能难以阅读
- 节目名称可能被截断

### 优化后
- 使用列表布局：`space-y-1 max-h-64 overflow-y-auto border rounded-md`
- 每个节目显示为一行，包含完整信息
- 更好的信息层次结构和可读性
- 支持滚动查看更多节目

## 界面改进详情

### 1. 布局结构
```tsx
// 优化前：网格卡片
<div className="grid grid-cols-3 sm:grid-cols-4 md:grid-cols-5 gap-2">
  <Button>...</Button>
</div>

// 优化后：列表行
<div className="space-y-1 max-h-64 overflow-y-auto border rounded-md">
  <div className="flex items-center justify-between p-3">...</div>
</div>
```

### 2. 信息展示
**优化后的每行包含：**
- **左侧**：节目序号（圆形徽章）+ 节目名称 + 参与者信息
- **右侧**：节目状态徽章 + 选中指示器

### 3. 视觉设计
- **选中状态**：`bg-primary/10 border-l-4 border-l-primary`
- **悬停效果**：`hover:bg-muted/50`
- **状态颜色**：
  - 等待中：`bg-yellow-100 text-yellow-700`
  - 进行中：`bg-blue-100 text-blue-700`
  - 已完成：`bg-green-100 text-green-700`

### 4. 交互体验
- 点击整行即可选择节目
- 清晰的选中状态指示
- 待保存状态用 `*` 标记
- 支持键盘导航

## 功能特性

### 1. 状态指示
- **节目序号徽章**：根据状态显示不同颜色
- **状态标签**：右侧显示当前状态
- **选中指示器**：蓝色圆点表示当前选中

### 2. 信息完整性
- **节目名称**：完整显示，支持长名称
- **参与者信息**：显示所有参与者姓名
- **状态变更提示**：未保存的状态变更用星号标记

### 3. 响应式设计
- 在不同屏幕尺寸下保持良好的可读性
- 自动滚动支持查看更多节目
- 合理的间距和字体大小

## 技术实现

### 修改文件
- `judge-system/src/components/dashboard/competitions/DisplayManagement.tsx`

### 关键代码变更
```tsx
// 节目列表容器
<div className="space-y-1 max-h-64 overflow-y-auto border rounded-md">
  {programs.map((program, index) => (
    <div
      key={program.id}
      className={`flex items-center justify-between p-3 hover:bg-muted/50 cursor-pointer transition-colors ${
        form.watch('currentProgramId') === program.id 
          ? 'bg-primary/10 border-l-4 border-l-primary' 
          : index > 0 ? 'border-t' : ''
      }`}
      onClick={() => form.setValue('currentProgramId', program.id)}
    >
      {/* 左侧：序号 + 信息 */}
      <div className="flex items-center space-x-3 flex-1 min-w-0">
        <div className="flex-shrink-0">
          <div className={`w-8 h-8 rounded-full flex items-center justify-center text-sm font-bold ${statusColors}`}>
            {program.order}
          </div>
        </div>
        <div className="flex-1 min-w-0">
          <div className="flex items-center space-x-2">
            <span className="font-medium text-sm truncate">{program.name}</span>
            {pendingStatusUpdates[program.id] && (
              <span className="text-xs text-amber-600">*</span>
            )}
          </div>
          <div className="text-xs text-muted-foreground truncate">
            {program.participants.map(p => p.name).join('、')}
          </div>
        </div>
      </div>
      
      {/* 右侧：状态 + 选中指示器 */}
      <div className="flex items-center space-x-2 flex-shrink-0">
        <div className={`px-2 py-1 rounded-full text-xs font-medium ${statusColors}`}>
          {getProgramStatusText(getProgramDisplayStatus(program.id))}
        </div>
        {form.watch('currentProgramId') === program.id && (
          <div className="w-2 h-2 bg-primary rounded-full"></div>
        )}
      </div>
    </div>
  ))}
</div>
```
