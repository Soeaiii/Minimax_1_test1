# 评委卡片大小调整功能

## 功能概述

在大屏幕管理页面中，现在可以动态调整评委卡片的大小和样式，包括：

- 卡片宽度
- 卡片内边距
- 卡片间距
- 头像大小
- 姓名字体大小
- 分数字体大小

## 使用方法

1. 进入比赛管理页面
2. 点击"大屏幕管理"
3. 在左侧设置面板中找到"评委卡片大小设置"部分
4. 调整各项参数：
   - **卡片宽度**: 200-500px，控制整个评委卡片的宽度
   - **卡片内边距**: 8-64px，控制卡片内部的间距
   - **卡片间距**: 10-80px，控制评委卡片之间的距离
   - **头像大小**: 80-300px，控制评委头像的尺寸
   - **姓名字体大小**: 12-36px，控制评委姓名的字体大小
   - **分数字体大小**: 18-60px，控制评委分数的字体大小
5. 点击"保存设置"应用更改
6. 可以点击"重置默认大小"恢复到默认设置

## 默认值

- 卡片宽度: 288px
- 卡片内边距: 32px
- 卡片间距: 40px
- 头像大小: 176px
- 姓名字体大小: 20px
- 分数字体大小: 36px

## 实时预览

设置更改后，可以通过"预览大屏幕"按钮在新窗口中查看效果。

## 技术实现

### 数据库字段

在 `DisplaySettings` 模型中添加了以下字段：

```prisma
judgeCardWidth      Int?      @default(288)
judgeCardPadding    Int?      @default(32)
judgeCardGap        Int?      @default(40)
judgeAvatarSize     Int?      @default(176)
judgeNameFontSize   Int?      @default(20)
judgeScoreFontSize  Int?      @default(36)
```

### 前端实现

- 管理界面使用数字输入框控制各项参数
- 大屏幕显示页面使用内联样式应用动态尺寸
- 支持实时预览和重置功能

### API 更新

- GET `/api/display/[competitionId]`: 返回包含卡片大小设置的配置
- PUT `/api/display/[competitionId]`: 更新卡片大小设置

## 注意事项

1. 只有在启用"显示裁判评分"时，卡片大小设置才会显示
2. 所有参数都有合理的范围限制，防止设置过大或过小的值
3. 设置会实时保存到数据库，并在大屏幕显示中立即生效
4. 头像的字体大小会根据头像尺寸自动调整，确保比例协调
