# 示例数据说明

本文档描述了种子脚本 `seed.ts` 创建的示例数据结构和内容。

## 运行种子脚本

```bash
npm run seed
# 或
pnpm seed
```

### 评分 (Scores)
为前3个已完成的节目创建了完整的评分：

- **评分数量**：36个评分记录（3个节目 × 3个评委 × 4个评分标准）
- **分数范围**：8.0-10.0分（随机生成）
- **评委**：每个节目都由3位评委按4个标准进行评分
- **评价**：每个评分都包含简单的文字评价

### 排名 (Rankings)
为已完成评分的3个节目生成了排名：

- 使用加权平均算法计算总分
- 根据评分标准权重计算最终得分
- 自动生成排名顺序

### 审计日志 (Audit Logs)

3. 评委提交评分的记录

## 数据特点

1. **完整性**：涵盖了系统所有主要实体和关系
2. **真实性**：使用符合中国文化背景的姓名和内容
3. **多样性**：包含不同类型的艺术表演和多种用户角色
4. **层次性**：从等待中到已完成的节目，展示了比赛的不同阶段
5. **关联性**：所有数据都有正确的关联关系，符合业务逻辑
