# 🏆 Match7 评审系统

一个专业的比赛评审管理系统，支持多种类型的比赛活动，包括文艺演出、技能竞赛、学术评比等。

## ✨ 功能特色

### 🎯 核心功能
- **多角色管理**：管理员、评委、参赛者三种角色，权限清晰分离
- **灵活评分**：支持自定义评分标准和权重设置
- **实时大屏**：专业的大屏幕展示系统，支持多种主题和实时数据更新
- **文件管理**：支持图片、视频等多媒体文件上传和管理
- **数据追踪**：完整的审计日志和操作记录
- **响应式设计**：适配各种屏幕尺寸，支持移动端访问

### 🚀 技术栈
- **前端框架**：Next.js 15.3.2 + React 19
- **UI 组件**：Tailwind CSS + shadcn/ui
- **数据库**：MongoDB + Prisma ORM
- **认证系统**：NextAuth.js
- **表单处理**：React Hook Form + Zod 验证
- **文件上传**：支持多种格式的文件上传

## 🚀 快速开始

### 环境要求
- Node.js 18.0+ 或 20.0+
- MongoDB 7.0+
- npm 9.0+ 或 pnpm 8.0+

### 安装步骤

1. **克隆项目**
```bash
git clone <项目地址>
cd match7/judge-system
```

2. **安装依赖**
```bash
npm install
# 或
pnpm install
```

3. **环境配置**
```bash
# 复制环境变量文件
cp .env.example .env

# 编辑 .env 文件，配置数据库连接等
```

4. **数据库初始化**
```bash
# 生成 Prisma 客户端
npx prisma generate

# 初始化种子数据
npm run prisma:seed
```

5. **启动应用**
```bash
# 开发模式
npm run dev

# 访问 http://localhost:3001
```
