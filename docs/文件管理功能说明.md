# 文件管理功能说明

## 功能概述

文件管理系统为比赛管理平台提供了完整的文件上传、存储、预览和下载功能。用户可以通过直观的界面管理各种类型的文件，包括图片、视频、音频和文档等。

## 主要功能

### 1. 文件上传
- **拖拽上传**：支持将文件直接拖拽到上传区域
- **点击上传**：点击上传区域选择文件
- **批量上传**：支持同时上传多个文件
- **文件类型检测**：自动识别文件类型并显示相应图标
- **文件大小限制**：默认最大文件大小为10MB
- **上传进度**：实时显示上传进度

### 2. 文件展示
- **网格视图**：以卡片形式展示文件，适合预览图片
- **列表视图**：以表格形式展示文件详细信息
- **文件预览**：图片文件支持缩略图预览
- **文件信息**：显示文件名、大小、上传时间等信息

### 3. 文件操作
- **文件下载**：支持单个文件下载
- **文件删除**：支持单个或批量删除文件
- **文件选择**：支持多选文件进行批量操作
- **全选功能**：一键选择所有文件

### 4. 搜索和过滤
- **文件名搜索**：根据文件名进行模糊搜索
- **类型过滤**：按文件类型（图片、视频、音频、文档）过滤
- **实时搜索**：输入即时显示搜索结果

### 5. 统计信息
- **总文件数**：显示系统中的文件总数
- **分类统计**：按类型统计文件数量
- **存储空间**：显示总的存储空间使用情况

## 技术实现

### 前端组件
- `FileUpload.tsx`：文件上传组件，支持拖拽和点击上传
- `FileGrid.tsx`：网格视图组件，以卡片形式展示文件
- `FileList.tsx`：列表视图组件，以表格形式展示文件
- `page.tsx`：主页面组件，整合所有功能

### 后端API
- `GET /api/files`：获取文件列表
- `POST /api/files/upload`：上传文件
- `DELETE /api/files`：批量删除文件
- `GET /api/files/[id]/download`：下载文件
- `GET /api/files/[id]/preview`：预览文件（仅图片）

### 数据库模型
```prisma
model File {
  id            String    @id @default(auto()) @map("_id") @db.ObjectId
  filename      String    // 原始文件名
  path          String    // 存储路径
  mimetype      String    // MIME类型
  size          Int       // 文件大小（字节）
  createdAt     DateTime  @default(now())
  
  // 关联关系
  programs      Program[] @relation("ProgramFiles")
  competitions  Competition[] @relation("CompetitionFiles")
}
```

## 文件存储

文件存储在 `public/uploads/` 目录下，文件名格式为：`原文件名_时间戳.扩展名`

例如：`document_1703123456789.pdf`

## 支持的文件类型

### 图片文件
- JPEG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)
- WebP (.webp)
- SVG (.svg)

### 视频文件
- MP4 (.mp4)
- WebM (.webm)
- AVI (.avi)
- MOV (.mov)

### 音频文件
- MP3 (.mp3)
- WAV (.wav)
- OGG (.ogg)
- AAC (.aac)

### 文档文件
- PDF (.pdf)
- Word (.doc, .docx)
- Excel (.xls, .xlsx)
- PowerPoint (.ppt, .pptx)
- 文本文件 (.txt)

## 使用说明

### 上传文件
1. 点击"上传文件"按钮打开上传对话框
2. 将文件拖拽到上传区域，或点击选择文件
3. 选择的文件会显示在待上传列表中
4. 点击"上传"按钮开始上传
5. 上传完成后文件会自动添加到文件列表中

### 管理文件
1. 使用搜索框根据文件名搜索文件
2. 使用类型过滤器按文件类型筛选
3. 切换网格/列表视图模式
4. 选择文件进行批量操作
5. 点击操作菜单下载或删除文件

### 文件关联
文件可以与节目和比赛进行关联，关联信息会显示在文件详情中。

## 安全考虑

1. **文件类型验证**：基于MIME类型验证文件类型
2. **文件大小限制**：防止上传过大文件
3. **文件名安全**：自动生成安全的文件名
4. **路径安全**：防止路径遍历攻击

## 性能优化

1. **分页加载**：大量文件时分页显示
2. **图片缩略图**：图片文件生成缩略图预览
3. **缓存策略**：静态文件设置适当的缓存头
4. **异步上传**：支持并发上传多个文件

## 故障排除

### 常见问题
1. **上传失败**：检查文件大小和类型限制
2. **预览不显示**：确保文件路径正确且文件存在
3. **下载失败**：检查文件权限和存储路径
4. **删除失败**：确保有足够的权限删除文件

### 日志查看
文件操作的错误信息会记录在服务器控制台中，可以通过查看日志来诊断问题。
