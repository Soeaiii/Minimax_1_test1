# 大屏幕评委管理功能说明

## 功能概述

大屏幕评委管理功能已从大屏幕显示页面移至大屏幕管理页面，管理员可以在管理页面中选择哪些评委的评分显示在大屏幕上，并且可以设置关闭自动刷新功能。

## 主要更新

### 1. 评委管理功能迁移 ⭐
- **从大屏幕显示页面移除**：移除了原有的控制面板和快捷键功能
- **集成到管理页面**：在大屏幕管理页面中添加了完整的评委选择功能
- **统一管理**：所有大屏幕相关设置现在都在管理页面中统一配置

### 2. 评委选择功能 ⭐
- **可视化选择**：显示所有评委列表，支持勾选/取消勾选
- **批量操作**：提供"全选"和"全不选"快速操作按钮
- **实时统计**：显示已选择评委数量和总评委数量
- **状态提示**：未选择评委时显示警告提示
- **头像显示**：显示评委头像和姓名，提升用户体验

### 3. 自动刷新控制 ⭐
- **可关闭设置**：管理员可以选择关闭大屏幕自动刷新
- **灵活配置**：支持设置刷新间隔或完全关闭自动刷新
- **实时评分优先**：由于已有实时评分功能，可以安全关闭自动刷新

### 4. 数据库模型更新
- **新增字段**：在 DisplaySettings 模型中添加 `selectedJudgeIds` 字段
- **数据持久化**：评委选择设置会保存到数据库中
- **向后兼容**：默认值为空数组，不影响现有数据

## 使用方法

### 访问评委管理
1. 进入比赛详情页面
2. 点击"大屏幕管理"按钮
3. 在显示设置中找到"评委选择"部分

### 配置评委显示
1. **启用评委评分显示**：确保"显示裁判评分"开关已开启
2. **选择评委**：
   - 在评委列表中勾选要显示的评委
   - 使用"全选"/"全不选"按钮快速操作
   - 查看选择统计（如"已选择 3/5"）
3. **保存设置**：点击"保存设置"按钮使配置生效

### 配置自动刷新
1. **关闭自动刷新**：将"自动刷新"开关设为关闭状态
2. **设置刷新间隔**：如果保持自动刷新，可以调整刷新间隔（1-60秒）
3. **保存设置**：点击"保存设置"按钮

## 技术实现

### 前端更新
- **管理页面增强**：在 `/dashboard/display/[competitionId]/page.tsx` 中添加评委选择组件
- **显示页面简化**：从 `/display/[competitionId]/page.tsx` 中移除控制面板相关代码
- **表单集成**：评委选择集成到现有的设置表单中

### 后端更新
- **数据库模型**：在 `DisplaySettings` 模型中添加 `selectedJudgeIds` 字段
- **API 支持**：更新显示设置 API 以支持保存和读取评委选择
- **实时评分**：SSE 端点已支持根据选择的评委过滤评分数据

### 数据流程
1. **管理页面**：管理员选择评委并保存设置
2. **数据库存储**：评委选择保存到 `DisplaySettings.selectedJudgeIds`
3. **实时传输**：SSE 根据选择的评委过滤并推送评分数据
4. **大屏幕显示**：只显示选中评委的评分信息

## 优势特点

### 1. 管理集中化
- **统一入口**：所有大屏幕设置都在管理页面中
- **权限控制**：只有管理员和组织者可以访问
- **操作简便**：一个页面完成所有配置

### 2. 用户体验优化
- **界面简洁**：大屏幕页面专注于显示，无多余控件
- **操作直观**：评委选择界面清晰易懂
- **实时反馈**：选择变化立即显示统计信息

### 3. 性能优化
- **减少刷新**：可以关闭自动刷新，依赖实时评分
- **精确过滤**：只传输选中评委的数据
- **资源节约**：避免不必要的页面刷新

### 4. 灵活配置
- **按需显示**：可以选择部分评委显示
- **动态调整**：随时修改评委选择
- **设置持久化**：配置自动保存

## 注意事项

1. **评委选择为空**：如果未选择任何评委，大屏幕将不显示评分数据
2. **实时评分依赖**：关闭自动刷新时，确保实时评分功能正常工作
3. **权限要求**：只有管理员和组织者可以修改大屏幕设置
4. **数据同步**：评委选择变更后需要保存设置才能生效

## 兼容性

- ✅ 向后兼容现有数据
- ✅ 不影响现有比赛
- ✅ 默认显示所有评委（如果未配置）
- ✅ 支持现有的实时评分功能

## 总结

这次更新将大屏幕的评委管理功能从显示页面迁移到了管理页面，实现了更好的功能分离和用户体验。管理员可以在统一的管理界面中配置所有大屏幕相关设置，包括评委选择和自动刷新控制，而大屏幕显示页面则专注于内容展示，界面更加简洁专业。
