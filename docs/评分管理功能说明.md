# 评分管理功能说明

## 功能概述

在节目管理的评分页面中，管理员和比赛组织者现在可以直接修改节目的评分数据。这个功能提供了完整的评分管理能力，包括添加、编辑和删除评分记录。

## 主要功能

### 1. 查看评分详情
- 显示所有评分标准的详细得分情况
- 按评分标准分组显示评分
- 显示每个标准的统计信息（平均分、最高分、最低分）
- 计算并显示总加权分数
- **总加权分数卡片始终显示**，即使没有任何评分也会显示"0.00"

### 2. 添加评分
- **多个添加评分入口**：
  - 总加权分数卡片右侧的"添加评分"按钮（始终可见）
  - 无评分时的"开始评分"按钮
- 选择评委和评分标准
- 输入分数（支持小数点后两位精度）
- 可选择添加评语

### 3. 编辑评分
- 点击评分记录旁的"更多操作"按钮
- 选择"编辑评分"
- 修改分数和评语（评委和评分标准不可修改）
- 保存修改

### 4. 删除评分
- 点击评分记录旁的"更多操作"按钮
- 选择"删除评分"
- 确认删除操作

## 界面优化

### 总加权分数卡片
- **始终显示**：无论是否有评分数据都会显示
- **动态内容**：
  - 有评分时：显示实际的加权平均分和"基于所有评分标准的加权平均分"
  - 无评分时：显示"0.00"和"暂无评分数据"
- **添加评分按钮**：右侧始终显示"添加评分"按钮，方便快速添加

### 无评分状态
- 显示友好的提示信息
- 提供"开始评分"按钮引导用户操作
- 保持界面的一致性和可用性

## 权限控制

- **管理员（ADMIN）**：可以对所有节目的评分进行增删改查
- **比赛组织者**：只能对自己组织的比赛中的节目评分进行增删改查
- **评委（JUDGE）**：只能查看评分，不能修改
- **普通用户**：只能查看评分，不能修改

## 使用步骤

### 添加评分
1. 进入节目详情页面
2. 切换到"评分"标签页
3. 点击以下任一按钮：
   - 总加权分数卡片右侧的"添加评分"按钮
   - 无评分时的"开始评分"按钮
4. 在弹出的对话框中：
   - 选择评委
   - 选择评分标准
   - 输入分数（大于等于0，支持小数点后两位）
   - 可选择输入评语
5. 点击"添加评分"保存

### 编辑评分
1. 在评分详情中找到要编辑的评分记录
2. 点击该记录右侧的"⋯"按钮
3. 选择"编辑评分"
4. 在弹出的对话框中修改分数和评语
5. 点击"更新评分"保存修改

### 删除评分
1. 在评分详情中找到要删除的评分记录
2. 点击该记录右侧的"⋯"按钮
3. 选择"删除评分"
4. 在确认对话框中点击"删除"

## 数据验证

- **分数范围**：分数必须大于等于0，无上限限制
- **分数精度**：支持小数点后最多两位（如：85.75）
- **必填字段**：评委、评分标准和分数为必填项
- **重复评分**：同一评委对同一标准的重复评分会自动更新现有记录

## 审计日志

所有评分操作都会记录在审计日志中：
- 创建评分：记录操作者、节目ID、评分标准和分数
- 更新评分：记录操作者、节目ID、评分标准、原分数和新分数
- 删除评分：记录操作者、节目ID、评分标准和被删除的分数

## 技术实现

### API端点
- `GET /api/programs/[id]/scores` - 获取节目评分
- `POST /api/programs/[id]/scores` - 添加/更新评分
- `DELETE /api/programs/[id]/scores?scoreId=[scoreId]` - 删除评分
- `GET /api/competitions/[id]/judges-and-criteria` - 获取评委和评分标准

### 组件结构
- `ScoreManager` - 评分管理主组件
- `ScoreDisplay` - 评分显示组件
- `ScoreEditDialog` - 评分编辑对话框

### 状态管理
- 使用React hooks管理本地状态
- 实时更新评分数据
- 错误处理和加载状态

## 注意事项

1. **权限检查**：每次操作都会进行权限验证
2. **数据一致性**：评分修改会自动重新计算总分和排名
3. **用户体验**：提供实时反馈和错误提示
4. **数据安全**：所有操作都有审计日志记录
5. **分数灵活性**：分数不受评分标准满分限制，可根据实际需要设置
6. **界面一致性**：总加权分数卡片始终显示，保持界面的连贯性
