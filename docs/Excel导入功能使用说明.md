# Excel导入功能使用说明

## 功能概述

比赛管理系统支持通过Excel表格批量导入选手信息和节目信息，大大提高了数据录入的效率。

## 支持的导入类型

### 1. 选手信息导入

**支持的文件格式：**
- Excel文件 (.xlsx, .xls)
- CSV文件 (.csv)

**支持的字段：**
- **姓名**（必填）：选手或团队名称
- **简介**（可选）：选手的个人介绍或专业背景
- **团队**（可选）：所属团队或院系
- **联系方式**（可选）：邮箱或电话号码

**字段识别规则：**
系统会自动识别列标题，支持多种命名方式：
- 姓名：姓名、选手姓名、参赛者姓名、name、名字
- 简介：简介、个人简介、介绍、bio、description
- 团队：团队、团队名称、所属团队、team、队伍
- 联系方式：联系方式、联系电话、电话、contact、phone、手机

### 2. 节目信息导入

**支持的字段：**
- **节目名称**（必填）：节目的名称
- **描述**（可选）：节目的详细描述
- **顺序**（可选）：节目在比赛中的演出顺序，如未填写则自动分配
- **选手姓名**（可选）：参与该节目的选手，支持多个选手用逗号分隔

**字段识别规则：**
- 节目名称：节目名称、节目、name、名称
- 描述：描述、节目描述、简介、description、desc
- 顺序：顺序、序号、排序、order、编号
- 选手姓名：选手姓名、参赛者、表演者、participants、选手

## 使用步骤

### 导入选手信息

1. **进入选手管理页面**
   - 在仪表板中点击"选手管理"
   - 点击页面右上角的"批量导入"按钮

2. **下载模板文件**
   - 在导入对话框中点击"下载模板"
   - 获取标准格式的Excel模板文件

3. **填写Excel文件**
   - 按照模板格式填写选手信息
   - 确保"姓名"列不为空
   - 其他字段可根据实际情况填写

4. **上传并预览**
   - 点击"选择Excel文件"按钮
   - 系统会自动解析文件并显示预览
   - 检查数据是否正确识别

5. **确认导入**
   - 查看有效数据和无效数据统计
   - 点击"导入"按钮完成批量导入

### 导入节目信息

1. **选择导入位置**
   - 在节目管理页面或比赛详情页面
   - 点击"批量导入"按钮

2. **选择目标比赛**
   - 在导入对话框中选择节目所属的比赛
   - 如果是在比赛详情页面导入，则会自动选择

3. **准备Excel文件**
   - 下载节目信息模板
   - 填写节目名称（必填）
   - 填写其他可选信息

4. **选手匹配规则**
   - 系统会自动匹配"选手姓名"列中的选手
   - 如果选手不存在，会自动创建新选手
   - 支持多个选手用逗号、顿号分隔

5. **完成导入**
   - 确认数据无误后点击导入
   - 系统会自动处理选手关联关系

## 数据验证规则

### 选手信息验证
- 姓名不能为空
- 不允许导入重复的选手（相同姓名和团队）
- 联系方式格式会进行基本验证

### 节目信息验证
- 节目名称不能为空
- 同一比赛内不允许重复的节目名称
- 顺序号必须为正整数（如果填写）
- 选手姓名会进行匹配验证

## 错误处理

### 常见错误类型
1. **文件格式错误**：不支持的文件类型
2. **必填字段缺失**：姓名或节目名称为空
3. **数据重复**：已存在相同的选手或节目
4. **字段格式错误**：数据类型不匹配

### 错误查看
- 导入预览中会显示无效数据
- 每行错误都有详细的错误说明
- 只有有效数据会被导入

## 最佳实践

1. **使用模板文件**
   - 总是从下载的模板开始
   - 保持列标题格式不变

2. **数据准备**
   - 在导入前清理数据，去除空行
   - 确保必填字段完整
   - 统一数据格式

3. **分批导入**
   - 大量数据建议分批导入
   - 每批建议不超过500条记录

4. **数据备份**
   - 导入前备份现有数据
   - 保存Excel源文件

## 技术支持

如遇到问题，请检查：
1. 文件格式是否正确
2. 必填字段是否完整
3. 数据是否符合验证规则

更多技术支持请联系系统管理员。
