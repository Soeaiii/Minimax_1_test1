# 大屏幕控制功能说明

## 功能概述

大屏幕显示页面现在支持评委选择和手动刷新功能，管理员可以控制哪些评委的评分显示在大屏幕上，并且可以手动刷新数据而不需要刷新整个页面。

## 主要功能

### 1. 评委选择控制 ⭐
- **选择性显示**：可以选择特定评委的评分显示在大屏幕上
- **实时过滤**：选择变化后立即生效，无需刷新页面
- **全选/全不选**：快速选择所有评委或清空选择
- **可视化界面**：显示评委头像、姓名和选择状态

### 2. 手动刷新功能 ⭐
- **手动控制**：可以手动刷新大屏幕数据
- **无页面刷新**：只刷新数据，不重新加载背景图片等资源
- **快捷键支持**：F5 键快速刷新
- **状态指示**：显示刷新进度状态

### 3. 控制面板
- **隐藏式设计**：默认隐藏，按快捷键显示
- **快捷键操作**：Ctrl+Shift+C 显示/隐藏控制面板
- **实时状态**：显示当前选择的评委数量
- **友好提示**：提供操作指导和状态提示

## 使用方法

### 访问控制面板
1. 打开大屏幕显示页面：`/display/[competitionId]`
2. 按 `Ctrl+Shift+C` 显示控制面板
3. 控制面板会出现在左上角

### 评委选择
1. 在控制面板中查看所有评委列表
2. 勾选/取消勾选要显示的评委
3. 使用"全选"/"全不选"按钮快速操作
4. 选择变化后实时生效

### 手动刷新
1. 点击控制面板中的"手动刷新大屏幕"按钮
2. 或者按 `F5` 键快速刷新
3. 刷新过程中会显示进度指示器

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+C` | 显示/隐藏控制面板 |
| `F5` | 手动刷新大屏幕数据 |

## 技术实现

### 后端 API 更新
- **SSE 端点增强**：支持评委过滤参数
- **用户 API 更新**：返回评委头像信息
- **实时过滤**：根据选择的评委ID过滤评分数据

### 前端组件
- **DisplayControlPanel**：控制面板组件
- **useScoreStream Hook**：支持评委选择参数
- **键盘事件处理**：快捷键支持

### 数据流程
```
用户选择评委 → 更新 Hook 参数 → 重新建立 SSE 连接 → 
服务器过滤数据 → 实时推送过滤后的评分 → 大屏幕更新显示
```

## 故障排除

### 常见问题

1. **控制面板不显示**
   - 检查快捷键是否正确（Ctrl+Shift+C）
   - 确认浏览器支持键盘事件

2. **评委选择不生效**
   - 检查网络连接
   - 查看浏览器控制台错误
   - 确认 SSE 连接状态

3. **手动刷新失败**
   - 检查 API 端点可用性
   - 确认比赛ID正确
   - 查看服务器日志

### 调试方法

1. **浏览器开发者工具**
   - Network 标签查看 SSE 连接参数
   - Console 标签查看错误信息
   - Elements 标签检查控制面板状态

2. **快捷键测试**
